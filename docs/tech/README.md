# Tech Docs

Dokumentasi teknis untuk `moneypath-ui`.

Dokumen di folder ini fokus ke hal yang sudah benar-benar ada di codebase, supaya tetap selaras dengan prinsip MVP-first dan tidak mendokumentasikan fitur yang belum dibangun.

## Daftar Dokumen

- `phase-0-foundation.md`  
  Ringkasan setup awal frontend: stack, struktur folder, routing, auth foundation, API foundation, dan cara menjalankan project.
- `phase-1-layout.md`  
  Ringkasan layout system awal: app shell, responsive header, mobile navigation, shared card pattern, dan spacing rules.
